# Genesis Template Project
A template to prove out the Retail Clerk game engine and build tooling.

Game engine features demonstrated:
- [x] Read controllers
    - [x] 6-button support
- [x] Load palettes
- [x] Load and draw tiles
- [x] Play background music
- [ ] Play sound effects (supported in sound driver, not in engine yet)
- [x] General game engine
    - [x] Move sprites
    - [x] Sprite z-order
    - [x] Load and draw scenes
    - [x] Collision
    - [x] Dialog management
    - [x] 2/4 choice selection dialogs
    - [x] Inventory management
    - [x] Title screen
    - [x] Menu screen with multiple sections
    - [x] Ending screens
    - [x] Pausing
- [x] Game event handling
    - [x] Enter/exit scenes
    - [x] Player selections
- [x] Run scripted events
    - [x] Dialogs
    - [x] Move sprites
    - [x] Change palettes    
    - [x] Change scenes
    - [x] Change font
    - [x] Change player sprite
    - [x] Delay timers
    - [x] End game

Build tool features demonstrated:
- [x] Build rom header
- [x] Build memory map and constants
- [x] Build lookup tables
- [ ] Create backups (supported but not used in this demo)
- [x] Build collision data from .png
- [x] Build scripted events from .json
- [x] Build palettes from .png
- [x] Build tiles from .png
    - [x] Basic tile deduplication
- [x] Build movement patterns from .json
- [x] Build sprites from .png
- [x] Build scenes from .json
- [x] Build text from .json
- [x] Build action table from .json
- [x] Convert .xm to .esf (audio format)
- [x] Compile and package
